This is the Micropolis Facebook Application Page Administration page.

You probably got here if you added the Micropolis application to your page, clicked the page's "Edit Info" link, then clicked the "Apps" tab, then clicked the Micropolis application's "Go to App" link.

What you expect to be able to do here, I have no idea.

But for what it's worth: The page id is "${fb_page_id}". The app id is "${app_id}". The user id is "${user_id or 'undefined'}". The user name is "${user_name}".
