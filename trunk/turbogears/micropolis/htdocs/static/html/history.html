History of Micropolis (SimCity)

Will Wright wrote the original SimCity city simulation game, first released in 1989. On January 10, 2008 the SimCity source code was released under the free software GPL 3 license under the name Micropolis.

DUX Software licensed the rights from Maxis to port SimCity to Unix in 1991. Don Hopkins started with the Mac version of the original SimCity Classic source code. He ported it to various Unix platforms, re-designed the user interface, created a multi-player version, and later adapted it for Linux and the OLPC XO-1 childrens' laptop.

He initially developed a version with PostScript graphics on the NeWS window system, using the HyperLook user interface development environment.

He subsequently developed a multi player version of SimCity for the X11 window system, using the TCL/Tk scripting language and user interface toolkit. It was shipped in 1993 and reviewed in Unix-World.

The multi player "SimCityNet" user interface was demonstrated to the HCI community in Amsterdam at the InterCHI'93 conference Interactive Experience, as described by the proposal, "SimCityNet: a Cooperative Multi User City Simulation".

Electronic Arts/Maxis recently relicensed this lineage of the SimCity Classic source code as Open Source software under GPL3. Don Hopkins adapted it to the OLPC and brought it up to EA's QA and branding requirements. Since it has been released, he has continued to develop the code for educational use.
