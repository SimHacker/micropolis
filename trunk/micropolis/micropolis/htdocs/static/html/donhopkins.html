Don Hopkins

Don Hopkins has worked with the SimCity source code for 16 years, ported to code to numerous platforms, and developed a colaborative multi player user interface to SimCity. He is committed to a long term vision of developing an open source educational software platform, and has written about his own and other peoples ideas on his web site.

He deeply appreciates the power of games to inspire and motivate kids to learn. As a high school student, he started using the ARPANET in 1982 to play Zork at the MIT-AI lab, and the orignal MUD (Multi User Dungeon) at Essex University. Those experiences motivated him to learn Lisp, Logo, Macsyma, Emacs and network programming, and exposed him to the work of Marvin Minsky, Seymoure Papert, Hal Abelson, Richard Stallman and Alan Kay.

Don enjoys writing fun exemplary software like games and user interfaces, that people can learn from and build on top of. The first educational game software he developed was a Logo version of "Adventure", which Terrapin Software shipped with the Commodode-64 version of Logo as an example of list processing.

Mark Weiser and Ben Shneiderman gave him the opportunity to learn user interface design and network programming at the University of Maryland Human Computer Interaction Lab. There he developed pie menus, hypermedia browsers and authoring tools, visual programming languages, extensibile scripting languages, and encountered SimCity.

James Gosling recruited him to work on the PostScript-based NeWS window system at Sun Microsystems, where he developed an Open Look user interface toolkit, an extensible tabbed window manager with pie menus, distributed object oriented applications, and multimedia programming examples like Pizzatool.

Will Wright trusted him to port SimCity to Unix, redesign the user interface and implement a networked multi player version of the game, which he demonstrated at INTERCHI'93 in Amsterdam, and shipped as a commercial product.

Will Wright later hired him into Maxis/Electronic Arts, to work on his core team that developed and shipped The Sims (the top selling PC game of all time). He designed and implemented the character animation system, content pipeline, user interface, visual programming and content creation tools.

After shipping The Sims, he worked as a contractor for Electronic Arts, developing easy-to-use tools to enable players to create their own Sims content, working on the Sims character animation tools, porting The Sims Online server to Linux.

Will Wright subsequently contracted him to work for his personal development lab, the Stupid Fun Club, programming robot brains, character simulation, talking toy prototypes, speech synthesis tools, and designing and developing software for reality TV productions and patent applications.

Don Hopkins has written ambitious but realistic and successful proposals for EA to "Develop Third Party Content Authoring Tools for The Sims" (which led to the thriving Sims player created content community), and to develop "Educational Multi Player SimCity for Linux" (which led to EA releasing Open Source SimCity under GPLv3 for the OLPC).

He has demonstrated Multi Player SimCity at ACM's InterCHI Conference, IBM's New Paradigms Workshop, the Exploratorium's Multimedia Playground, Interval's Electric Carnival, BayCHI at Xerox PARC, and MIT Media Lab's corporate sponsors meeting. Audiences are consistently excited about the possibilities of using SimCity educationally.

The "Open Sourcing" of SimCity makes the Micropolis Eduverse project possible and practical. Most of the hard work has already been done, and a playable online game is now low hanging fruit, easily attainable with open source technology, as this proof of concept demonstrates.

Don Hopkins draws on his work as a research programmer as well as a commercial product developer on many projects:

  • Gosling's Emacs distributed user interface programming, for Mike Gallaher at UniPress Software.
  • ScriptX multimedia programming and programming example design at Kaleida Labs (a joint venture of Apple and IBM).
  • Chuck Blanchard's Body Electric / Bounce, for David Levit at Levity Systems (a real time visual programming language for virtual reality and multimedia simulations, developed by VPL, used by Jaron Lanier).
  • Arthur van Hoff's HyperLook at the Turing Institute (a user interface development environment he used to port SimCity to SunOS).
  • Brad Myers' Garnet at Carnegie Mellon University Computer Science Department (a user interface management system written in CMU Common Lisp).
  • Multimedia Lisp programming, component software design, and social software research and development at Interval Research Corporation (Paul Alan's think tank).
  • ConnectedTV handheld and client/server architecture and implementation, at ConnectedMedia with David Levitt (an integrated universal IR remote control and personalized TV guide for the Palm).
  • OpenLaszlo development at Laszlo Systems (applications and components for a web based user interface programming language).
  • Project Alive! at Block Dietary Data Systems (a web site and Facebook application for nutritional analysis, coaching and behavior modification).
  • Python web server programming at Bartlett Net Services (Python, Zope, Plone and open source modules).
  • Social networking applications and web service APIs at Marc Canter's Broadband Mechanics.
  • Real time video, embedded device programming and Python virtual world authoring tools for Kennedy Labs.
  • Working on the eBook reader for the OLPC at the Internet Archive.
  • Integrating Gnu Radio with Python and adapting SimCity to the OLPC for John Gilmore.
  • Developing TomTom HOME for TomTom (a content management system written with Mozilla xulrunner, supporting user created content sharing and BitTorrent downloading).
